# Assignment Overview

This repository contains the solutions for **Assignment 1** of the Computer Programming course.

The purpose of this assignment is to strengthen the understanding of C++ decision-making structures by implementing each problem using five different control structures:

- if
- if-else
- if-else-if
- switch
- nested if

Each problem is solved separately to demonstrate the use and differences of each control structure.
